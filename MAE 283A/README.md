# MAE 283A Parametric Identification: Theory and Methods 

MAE 283A covers content ranging from constructing dynamical models from experimental data, deterministic and stochastic discrete time signals, discrete time systems, and nonparametric identification; correlation and spectral analysis. Along with parametric identification; realization and prediction error methods, least squares estimation, approximate modeling, experiment design, and frequency domain identification.

# Final Project
The final project for this course had students analyze unique experiments and build models to represent the data. For my project I analyzed data from a microgrid system undergoing a step excitation experiment. The experiment involved a Princeton Power Systems BIGI 250 inverter inside the Kaiser Permenante Microgrid. An inverter connected to a Battery energy storage system commanded several step changes in the real power demand and phasor measurement units measured the dynamical response of the BESS. Data analytics was performed in MATLAB and a 3rd order Box Jenkins model was developed to approximate the system. The report and code used to develop the model are contained in this folder.

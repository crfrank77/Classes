# ECE 128B Renewable Energy and Power Grid Modernization

ECE 128B is an in-depth course focued on the future power grids, covering the practical aspects of the technologies, their design, and system implementation. Topics include the changing nature of the grid with renewable resources, smart meters, synchrophasors (PMU), microgrids, distributed energy resources, and the associated information and communications infrastructures.

# Course Project
This course featured a 2 phase project that utilized the simulation software PSIM to build and test a Maximum Power Point Tracking controller for a Solar and Wind power generation setup that was tied to a Battery through a 3 phase Buck-Boost converter. Both phases reports and results are in this folder, along with the PSIM project files for each phase of the project.
